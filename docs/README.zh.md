### 安装

通过 OpenCode 使用一条命令：

```bash
opencode plugin @loyslow/opencode-voice-groq
```

安装后重启 OpenCode。首次启动将要求提供 Groq API 密钥。

可选的 CLI 安装程序：

```bash
npx @loyslow/opencode-voice-groq install
```
